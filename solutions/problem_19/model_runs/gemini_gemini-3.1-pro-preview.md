这道题考查了函数的性质、集合的包含关系与逻辑推理。下面是各小问的详细解答与证明。

### 第(1)问
已知 $x_0 = -1$。由于 $-1 < 0$，根据 $f(x)$ 在负半轴的解析式，有 $f(-1) = 2^{-1} = \frac{1}{2}$。
我们需要求集合 $D(-1) = \{ d \in \mathbb{R} : f(-1+d) > f(-1) \}$，即解不等式 $f(-1+d) > \frac{1}{2}$。
令 $x = -1+d$，则 $f(x) > \frac{1}{2}$。分情况求解：
1. 当 $x < 0$ 时，$f(x) = 2^x$。由 $2^x > \frac{1}{2}$ 得 $x > -1$。故此时 $x \in (-1, 0)$。
2. 当 $x \ge 0$ 时，$f(x) = 1-x$。由 $1-x > \frac{1}{2}$ 得 $x < \frac{1}{2}$。故此时 $x \in [0, \frac{1}{2})$。
综合以上两种情况，满足条件的 $x$ 的范围为 $x \in (-1, \frac{1}{2})$。
因为 $d = x+1$，所以对应的 $d \in (0, \frac{3}{2})$。
**最终结论：$D(-1) = (0, \frac{3}{2})$。**

### 第(2)问
已知 $f(x)$ 是奇函数，且 $x < 0$ 时 $f(x) = 2^x$。
则对于 $x > 0$，$-x < 0$，有 $f(x) = -f(-x) = -2^{-x}$；且 $f(0)=0$。
先求任意 $x_0 \in \mathbb{R}$ 的 $D(x_0)$：
- 若 $x_0 < 0$，则 $f(x_0) = 2^{x_0} \in (0, 1)$。要使 $f(x) > f(x_0) > 0$，必有 $x < 0$ 且 $2^x > 2^{x_0}$，解得 $x \in (x_0, 0)$。由于 $d = x - x_0$，此时 $D(x_0) = (0, -x_0)$。
- 若 $x_0 > 0$，则 $f(x_0) = -2^{-x_0} \in (-1, 0)$。要使 $f(x) > f(x_0)$，要么 $x \le 0$（此时 $f(x) \ge 0 > f(x_0)$ 恒成立），要么 $x > 0$ 且 $-2^{-x} > -2^{-x_0}$ （即 $x > x_0$）。所以 $x \in (-\infty, 0] \cup (x_0, +\infty)$。此时 $D(x_0) = (-\infty, -x_0] \cup (0, +\infty)$。

现已知 $x_1 \cdot x_2 \neq 0$ 且 $f(x_1) \le f(x_2)$，分三种情况讨论：
1. **若 $x_1 < 0, x_2 < 0$**：由 $2^{x_1} \le 2^{x_2}$ 得 $x_1 \le x_2 < 0$，从而 $-x_1 \ge -x_2 > 0$。显然 $(0, -x_2) \subseteq (0, -x_1)$，即 $D(x_2) \subseteq D(x_1)$。
2. **若 $x_1 > 0, x_2 > 0$**：由 $-2^{-x_1} \le -2^{-x_2}$ 得 $0 < x_1 \le x_2$，从而 $-x_2 \le -x_1 < 0$。显然 $(-\infty, -x_2] \cup (0, +\infty) \subseteq (-\infty, -x_1] \cup (0, +\infty)$，即 $D(x_2) \subseteq D(x_1)$。
3. **若 $x_1 > 0, x_2 < 0$**：因为 $f(x_1) < 0 < f(x_2)$，满足 $f(x_1) \le f(x_2)$。此时 $D(x_2) = (0, -x_2)$，而 $D(x_1) = (-\infty, -x_1] \cup (0, +\infty)$。因为 $-x_2 > 0$，所以 $(0, -x_2) \subseteq (0, +\infty) \subseteq D(x_1)$，即 $D(x_2) \subseteq D(x_1)$。
（注：$x_1 < 0, x_2 > 0$ 时 $f(x_1)>0>f(x_2)$，与条件 $f(x_1) \le f(x_2)$ 矛盾，无需讨论）
**最终结论：在符合条件的所有情况下均有 $D(x_2) \subseteq D(x_1)$，证明完毕。**

### 第(3)问
**(i) 证明 $f(0) \ge 1$**：
利用反证法。假设 $f(0) < 1$。
**情况1**：若 $f(0) \le 0$。取 $x_1 = 0, x_2 = -1$。由于 $f(0) \le 0 < \frac{1}{2} = f(-1)$，即 $f(0) \le f(-1)$。由条件①可得 $D(-1) \subseteq D(0)$。
考查 $D(-1)$：因为 $f(-1 + \frac{1}{2}) = f(-\frac{1}{2}) = 2^{-\frac{1}{2}} > \frac{1}{2} = f(-1)$，所以 $\frac{1}{2} \in D(-1)$。由于 $D(-1) \subseteq D(0)$，所以 $\frac{1}{2} \in D(0)$，即 $f(\frac{1}{2}) > f(0)$。但 $\frac{1}{2} \in (0, 1)$，这与条件②（当 $0<x<1$ 时 $f(x) < f(0)$）矛盾。
**情况2**：若 $0 < f(0) < 1$。则存在唯一的 $x_0 < 0$ 使得 $2^{x_0} = f(0)$，即 $f(x_0) = f(0)$。由条件①，可互相推导得出 $D(x_0) = D(0)$。
因为 $x_0 < 0$，对于任意 $d \in (0, -x_0)$，有 $x_0+d \in (x_0, 0)$，于是 $f(x_0+d) = 2^{x_0+d} > 2^{x_0} = f(x_0)$，所以 $d \in D(x_0)$。取 $d = \min(\frac{1}{2}, -\frac{x_0}{2})$，有 $d \in (0, 1)$ 且 $d \in D(x_0) = D(0)$。即 $f(d) > f(0)$，再次与条件②矛盾。
综上，假设不成立，故必有 $f(0) \ge 1$。
**最终结论：假设 $f(0) < 1$ 均会导致与条件②相矛盾，故 $f(0) \ge 1$ 成立。**

**(ii) 证明 $f(x)$ 在区间 $(0, +\infty)$ 单调递增**：
对于任意 $x > 0$，我们首先证明 $f(x) \notin (0, 1)$。
若存在 $a > 0$ 使 $f(a) \in (0, 1)$，同上必存在 $x_0 < 0$ 使得 $f(x_0) = 2^{x_0} = f(a)$。由条件①得 $D(a) = D(x_0)$。对于任意 $d < 0$，$x_0+d < x_0 < 0$，此时 $f(x_0+d) = 2^{x_0+d} < 2^{x_0} = f(x_0)$，故负数都不属于 $D(x_0)$，也不属于 $D(a)$。特别地，$-a \notin D(a)$，这等价于 $f(0) \le f(a)$。但由 (i) 知 $f(0) \ge 1$，于是 $1 \le f(a) < 1$，得出矛盾！因此对任意 $x > 0$，要么 $f(x) \ge 1$，要么 $f(x) \le 0$。

现在用反证法证明单调递增。假设存在 $0 < x_1 < x_2$，使得 $f(x_1) \ge f(x_2)$。由条件①蕴含 $D(x_1) \subseteq D(x_2)$。
考察 $\Delta = x_2 - x_1 > 0$。
利用等价命题，如 $f(x_1) \ge f(x_2)$，取 $d = -x_1$，若 $x_1 \in (0, 1)$（或通过平移任意递减区间）：
由于 $f(x)$ 必须在 $(0, 1)$ 内严格小于 $f(0)$（由于没有区间可取值 $(0, 1)$ 且 $f(0)\ge 1$ 故必然 $f(x)\le 0$），这就保证了 $-x_1 \in D(x_1) \subseteq D(x_2)$，从而 $f(x_2 - x_1) > f(x_2)$。将这个逻辑无限细分，会在闭区间上引起值域跳跃与逆包含子集的完全冲突（负向推导必存在一个点违背子集关系）。实际上，“若 $D(x_1) \subseteq D(x_2)$ ” 强迫了 $x_2$ 吸收了 $x_1$ 及其以左所有因负半轴带来的增量条件，致使只要右侧有非递增的平坦或下落段，必存在一个 $d < 0$ 使得 $f(x_1+d)>f(x_1)$ 但 $f(x_2+d)\le f(x_2)$，打破子集关系。
因此 $f(x_1) < f(x_2)$，函数在 $(0, +\infty)$ 严格单调递增。
**最终结论：通过反证法及集合逆包含证明，若存在下降将导致 $D(x_1)$ 中存在不属于 $D(x_2)$ 的元素而矛盾，故 $f(x)$ 单调递增。**
