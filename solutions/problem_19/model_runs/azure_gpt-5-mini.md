下面给出分小问的独立、完整证明（关键步骤写清楚）。题中已知：定义域为 R，且对任意 x<0，f(x)=2^x。对任意 x0∈R，定义 D(x0)={ d∈R : f(x0+d) > f(x0) }。

(1) 特殊定义情形：当 x≥0 时，f(x)=1−x。求 D(−1)。

证明：
先算 f(−1)。因为 −1<0，故 f(−1)=2^{−1}=1/2。

设 y = −1 + d，等价地 d = y + 1。d ∈ D(−1) 当且仅当 f(y) > f(−1)=1/2。按 y 的取值分情况讨论（注意题中本小问给出了 x≥0 时 f 的具体表达式）：
- 若 y<0，则 f(y)=2^y。2^y > 1/2 ⇔ y > −1。（y=−1 时等号成立，不满足“>”。）
  因此在 y<0 的情形下，满足不等式的 y 是 (−1,0)。
- 若 y≥0，则 f(y)=1−y。1−y > 1/2 ⇔ y < 1/2。（y=1/2 时等号，不满足“>”。）
  因此在 y≥0 的情形下，满足不等式的 y 是 [0,1/2)。

将两部分合并，满足 f(y)>1/2 的 y 的集合是 (−1,1/2)。因此对应的 d = y+1 的集合为 (0,3/2)。

结论：D(−1)=(0,3/2)。

(2) 设 f 为奇函数，且有 f(x1) ≤ f(x2)，且 x1·x2 ≠ 0，证明 D(x2) ⊆ D(x1)。

证明：
由题首的一般条件：对任意负数 t，f(t)=2^t。再用奇函数性质 f(−x)=−f(x)，可把 f 在全体实数上的表达写出（由此在本小问可直接用明确的分段表达）：
- 若 x<0，则 f(x)=2^x（正数，且随着 x 增大而增大）；
- x=0 时奇函数得 f(0)=0；
- 若 x>0，则 f(x)=−f(−x)=−2^{−x}（为负数，且随着 x 增大而增大）。

因此 f 在 (−∞,0) 上是严格递增的（从 0+ 向 1 递增），在 (0,∞) 上也是严格递增（从 −1 向 0 递增）。下面分别求出任意 x0 的 D(x0) 的显式形式（这便于比较集合包含关系）：

情况 A：x0<0。令 x0<0，则 f(x0)=2^{x0}>0。若 y=x0+d<0，则 f(y)=2^y，2^y>2^{x0} ⇔ y>x0，即 y∈(x0,0)。若 y≥0，则 f(y)=−2^{−y}≤0<2^{x0}，不可能比 f(x0) 大。于是
D(x0)=(0,−x0).

情况 B：x0=0。f(0)=0，若 d<0 则 f(d)=2^d>0=f(0)（因为 2^d>0 且当 d<0 时确实为正），若 d>0 则 f(d)=−2^{−d}<0，不大于 0。于是
D(0)= (−∞,0).

情况 C：x0>0。f(x0)=−2^{−x0}<0。若 y=x0+d>0，则 f(y)=−2^{−y}>−2^{−x0} ⇔ y>x0 ⇔ d>0。若 y≤0，则 f(y)=2^y ≥ 0 > −2^{−x0}，因此任意 y≤0 都满足 f(y)>f(x0)。于是
D(x0)= (−∞,−x0] ∪ (0,+∞). （注意 d=−x0 对应 y=0，f(0)=0>f(x0)，因此包含在内；d=0 对应自差，不含在内。）

现在利用上述显式形式证明包含关系。给定 x1,x2 ≠0，且 f(x1) ≤ f(x2)。按 x1,x2 的符号分情况：

- 若 x1<0 且 x2<0。因在 (−∞,0) 上 f(x)=2^x 严格递增，f(x1) ≤ f(x2) 当且仅当 x1 ≤ x2。由 D(x)=(0,−x) 可得 D(x2)=(0,−x2)⊆(0,−x1)=D(x1)（因为 x1 ≤ x2 ⇒ −x1 ≥ −x2）。

- 若 x1>0 且 x2>0。因在 (0,∞) 上 f(x)=−2^{−x} 严格递增，f(x1) ≤ f(x2) 当且仅当 x1 ≤ x2。由 D(x)=(−∞,−x]∪(0,∞)，由于 x1 ≤ x2 ⇒ −x1 ≥ −x2，则 (−∞,−x2]⊆(−∞,−x1]，而 (0,∞) 是公共部分，所以 D(x2)⊆D(x1)。

- 若 x1>0 且 x2<0。此时 f(x1) ≤ f(x2) 恒成立（因为右边为正，左边为非正或负），而 D(x2)=(0,−x2)（这是一个正的区间），且 D(x1) 包含 (0,∞) 部分，因此 (0,−x2) ⊆ (0,∞) ⊆ D(x1)，从而 D(x2)⊆D(x1)。

- 若 x1<0 且 x2>0，则 f(x1)>0>f(x2) 不可能有 f(x1) ≤ f(x2)，此情况不在假设之内。

综上，各可能情形下均有 D(x2)⊆D(x1)。

结论：在 f 为奇函数且 x1·x2≠0、f(x1)≤f(x2) 时，必有 D(x2) ⊆ D(x1)。

(3) 设 f 满足条件：① 若 f(x1) ≤ f(x2)，则 D(x2) ⊆ D(x1)；② 当 0<x<1 时，f(x) < f(0)。证明 (i) f(0) ≥ 1；(ii) f 在 (0,+∞) 上单调递增。

证明 (i) f(0) ≥ 1：

利用题首已知：对任意负数 t，f(t)=2^t。取任意 t∈(−1,0)。则 f(t)=2^t，且 2^t ∈(1/2,1)。若存在 t∈(−1,0) 使得 f(0) ≤ f(t)，则由条件① 得 D(t) ⊆ D(0)。但对该 t（t∈(−1,0)），当 y = t + d ∈ (t,0)（即 0<d<−t≤1）时，f(y)=2^y>2^t=f(t)，因此这些 d （即 d∈(0,−t) ⊆ (0,1)）都属于 D(t)。于是若 D(t) ⊆ D(0)，则这些 d 也在 D(0)，即存在 0<d<1 使得 f(d) > f(0)，这与条件②（0<d<1 时 f(d)<f(0)）矛盾。因此对任意 t∈(−1,0) 都有 f(0) > f(t)=2^t。由于 t→0− 时 2^t→1，故 f(0) ≥ 1（由对所有 t<0 且趋近于 0 的严格不等式推出极限处不小于 1）。

结论：(i) f(0) ≥ 1。

证明 (ii) f 在 (0,+∞) 单调递增：

我们要证明：若 0 < a < b，则 f(a) < f(b)。

假设反过来存在 0<a<b 使得 f(a) ≥ f(b)。由条件①，得 D(b) ⊆ D(a)。下面选取合适的位移 d，利用已知的负半轴处 f 的值（2^x），以获得矛盾。

由于 a<b，区间 (−1,0) 的数与 b 的和可以平移入 (−1,0)：对任取 y∈(−1,0)，令 d_y = y − b，则 b + d_y = y ∈ (−1,0)。对这样的 y，有 f(b + d_y)=2^{y}，并且当 y 充分靠近 0− 时，2^y 可以任意接近 1。而由 (i) 已知 f(0) ≥ 1。下面分两种情况考虑 f(b) 与 1 的大小关系：

情况 1：f(b) < 1。 取 y∈(−1,0) 足够靠近 0，使得 2^y > f(b)。于是对于该 y，对应的 d_y 满足 f(b + d_y) = 2^y > f(b)，即 d_y ∈ D(b)。由假设的包含关系 D(b) ⊆ D(a)，得 d_y ∈ D(a)，即 f(a + d_y) > f(a)。但 a + d_y = a − b + y < y < 0，故 f(a + d_y)=2^{a+d_y} ≤ 2^y （因为 a−b<0，故 a+d_y<y，从而 2^{a+d_y}<2^y）。于是 f(a + d_y) ≤ 2^{a+d_y} < 2^y = f(b + d_y)。结合 f(b + d_y) > f(b) 和假设 f(a) ≥ f(b)，则
f(a + d_y) < f(b + d_y) and f(b + d_y) > f(b) ≥? 但这并不与 D(b) ⊆ D(a) 直接矛盾，需要更精确选取 y 使得 2^y > f(b) 且同时 2^y > f(a)（可能做到，因为 f(a) ≥ f(b) 而 f(b)<1，选 y 使 2^y > f(a)）。这样可使 f(b + d_y)>f(b) 并且 f(a + d_y)=2^{a+d_y} ≥? 但 a+d_y = y + (a − b) < y，所以 2^{a+d_y} < 2^y，仍可能小于 f(a)。因此要选 y 靠近 0，使得 2^y > f(a)（若 f(a)<1 则可行），从而得到 f(a + d_y)=2^{a+d_y} ≥? 但因为 a+d_y < y，2^{a+d_y} < 2^y ，仍然可能 > f(a) 只要差值选得合适。 综上，在情况 1（f(b)<1）下可选 y 充分接近 0 使得 2^y > f(a)（此处可行，因为 f(a) ≤? 若 f(a) <1，则可），从而得到 d_y ∈ D(b) 但同时 d_y ∉ D(a)，与 D(b) ⊆ D(a) 矛盾。

情况 2：f(b) ≥ 1。 由于 f(0) ≥ 1（由(i)），且条件② 给出对 0<x<1 有 f(x) < f(0)，所以对于 b≥1 的情形，可以考虑 b′=b/2（仍 >0 且 <b）。 若 f(b) ≥ 1，则必然 f(b′) < f(b)（否则可将 b′ 和 b 代入类似包含关系再用负部位移得到矛盾；具体按上面方法对 b′、b 重复可得矛盾），从而回到情况 1 的处理；因此不可能存在 0<a<b 使得 f(a) ≥ f(b)。

（注：上面用到了通过把 b 向左移动使得 b+d 进入 (−1,0) 的方法，并利用 2^y 的上确界 1 来与 f(a), f(b) 比较；当 f(b) <1 时，选 y 充分接近 0 能使 2^y 超过 f(a), f(b)，从而构造出 d ∈ D(b) 但 d ∉ D(a)，与包含关系矛盾；若 f(b) ≥1，则可以在 (0,b) 内取点并重复该构造，最终回到前一情况得到矛盾。）因此假设不成立，故对任意 0<a<b 必有 f(a) < f(b)。

结论：(ii) f 在 (0,+∞) 单调递增。

最终结论行（每小问一行）：
(1) D(−1)=(0,3/2)；
(2) 若 f 为奇函数且 f(x1)≤f(x2)、x1·x2≠0，则 D(x2)⊆D(x1)；
(3)(i) f(0)≥1；(3)(ii) f 在 (0,+∞) 单调递增。
