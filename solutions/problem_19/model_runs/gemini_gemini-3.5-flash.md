### 解题过程

#### (1) 解：
由题意，当 $x \ge 0$ 时，$f(x) = 1-x$；当 $x < 0$ 时，$f(x) = 2^x$。
所以函数 $f(x)$ 的解析式为：
$$f(x) = \begin{cases} 2^x, & x < 0 \\ 1-x, & x \ge 0 \end{cases}$$
我们要计算 $D(-1) = \{ d \in \mathbb{R} : f(-1+d) > f(-1) \}$。
因为 $x_0 = -1 < 0$，所以 $f(-1) = 2^{-1} = \frac{1}{2}$。
设 $x = -1+d$，则不等式 $f(-1+d) > f(-1)$ 即为 $f(x) > \frac{1}{2}$。
我们分类讨论求解该不等式：
1. 当 $x < 0$ 时，由 $2^x > \frac{1}{2} = 2^{-1}$ 得 $x > -1$，结合 $x < 0$，解得 $x \in (-1, 0)$；
2. 当 $x \ge 0$ 时，由 $1-x > \frac{1}{2}$ 得 $x < \frac{1}{2}$，结合 $x \ge 0$，解得 $x \in \left[0, \frac{1}{2}\right)$。

综上所述，不等式 $f(x) > \frac{1}{2}$ 的解集为 $x \in \left(-1, \frac{1}{2}\right)$。
因为 $d = x + 1$，所以：
$$d \in \left(0, \frac{3}{2}\right)$$
故 $D(-1) = \left(0, \frac{3}{2}\right)$。

#### (2) 证明：
因为 $f(x)$ 是定义在 $\mathbb{R}$ 上的奇函数，且当 $x < 0$ 时，$f(x) = 2^x$。
所以当 $x > 0$ 时，$-x < 0$，由奇函数性质有：
$$f(x) = -f(-x) = -2^{-x}$$
且 $f(0) = 0$。因此 $f(x)$ 的解析式为：
$$f(x) = \begin{cases} 2^x, & x < 0 \\ 0, & x = 0 \\ -2^{-x}, & x > 0 \end{cases}$$
由解析式可知，当 $x < 0$ 时，$f(x) \in (0, 1)$ 且严格单调递增；当 $x > 0$ 时，$f(x) \in (-1, 0)$ 且严格单调递增。
已知 $x_1 \cdot x_2 \ne 0$ 且 $f(x_1) \le f(x_2)$。
因为 $x_1, x_2$ 均不为 $0$，根据 $f(x)$ 的值域分布，我们对 $x_1, x_2$ 的正负进行分类讨论：
- 若 $x_1 \le x_2 < 0$，此时 $f(x_1) \le f(x_2)$ 显然成立；
- 若 $0 < x_1 \le x_2$，此时 $f(x_1) \le f(x_2)$ 显然成立；
- 若 $x_1 > 0$ 且 $x_2 < 0$，此时 $f(x_1) \in (-1, 0)$，$f(x_2) \in (0, 1)$，有 $f(x_1) < f(x_2)$ 成立。
（注：由于当 $x > 0$ 时 $f(x) < 0$，当 $x < 0$ 时 $f(x) > 0$，因此不存在 $x_1 < 0 < x_2$ 满足 $f(x_1) \le f(x_2)$ 的情况）。

现在我们先求出任意 $x_0 \ne 0$ 时集合 $D(x_0)$ 的表达式。
因为 $D(x_0) = \{ d \in \mathbb{R} : f(x_0+d) > f(x_0) \}$，设 $x = x_0+d \iff d = x-x_0$，则：
- **情况 A**：若 $x_0 < 0$，此时 $f(x_0) = 2^{x_0} \in (0, 1)$。
  若要 $f(x) > f(x_0)$：
  1. 当 $x \ge 0$ 时，$f(x) \le 0 < f(x_0)$，无解；
  2. 当 $x < 0$ 时，$2^x > 2^{x_0} \iff x > x_0$，结合 $x < 0$ 得 $x \in (x_0, 0)$。
  所以 $x \in (x_0, 0)$，即 $d = x-x_0 \in (0, -x_0)$。故 $D(x_0) = (0, -x_0)$。

- **情况 B**：若 $x_0 > 0$，此时 $f(x_0) = -2^{-x_0} \in (-1, 0)$。
  若要 $f(x) > f(x_0)$：
  1. 当 $x < 0$ 时，$f(x) = 2^x > 0 > f(x_0)$，恒成立，此时 $x \in (-\infty, 0)$；
  2. 当 $x = 0$ 时，$f(0) = 0 > f(x_0)$，成立；
  3. 当 $x > 0$ 时，$-2^{-x} > -2^{-x_0} \iff 2^{-x} < 2^{-x_0} \iff x > x_0$，此时 $x \in (x_0, +\infty)$。
  综上， $x \in (-\infty, 0] \cup (x_0, +\infty)$，从而 $d = x-x_0 \in (-\infty, -x_0] \cup (0, +\infty)$。故 $D(x_0) = (-\infty, -x_0] \cup (0, +\infty)$。

下面针对上述三种情形分别证明 $D(x_2) \subseteq D(x_1)$：
1. **第一种情况：$x_1 \le x_2 < 0$**。
   此时 $D(x_1) = (0, -x_1)$，$D(x_2) = (0, -x_2)$。
   因为 $x_1 \le x_2 < 0$，所以 $-x_1 \ge -x_2 > 0$。
   显然 $(0, -x_2) \subseteq (0, -x_1)$，即 $D(x_2) \subseteq D(x_1)$。

2. **第二种情况：$0 < x_1 \le x_2$**。
   此时 $D(x_1) = (-\infty, -x_1] \cup (0, +\infty)$，$D(x_2) = (-\infty, -x_2] \cup (0, +\infty)$。
   因为 $0 < x_1 \le x_2$，所以 $-x_1 \ge -x_2$。
   显然 $(-\infty, -x_2] \subseteq (-\infty, -x_1]$，故 $D(x_2) \subseteq D(x_1)$。

3. **第三种情况：$x_1 > 0$ 且 $x_2 < 0$**。
   此时 $D(x_1) = (-\infty, -x_1] \cup (0, +\infty)$，$D(x_2) = (0, -x_2)$。
   对任意 $d \in D(x_2)$，有 $0 < d < -x_2$，从而 $d \in (0, +\infty)$。
   因为 $(0, +\infty) \subseteq D(x_1)$，所以 $d \in D(x_1)$。
   因此 $D(x_2) \subseteq D(x_1)$。

综上所述，若 $f(x)$ 是奇函数，且 $f(x_1) \le f(x_2)$，其中 $x_1 \cdot x_2 \ne 0$，则 $D(x_2) \subseteq D(x_1)$ 成立。

#### (3) 证明：
##### (i) 证明：$f(0) \ge 1$。
采用反证法。假设 $f(0) < 1$。
因为当 $x < 0$ 时，$f(x) = 2^x$ 严格单调递增，且其在 $(-\infty, 0)$ 上的值域为 $(0, 1)$。
由于 $f(0) < 1$ 且 $\lim_{x \to 0^-} 2^x = 1$，所以在区间 $(-\infty, 0)$ 内必定存在某个 $x_0$ 使得 $f(x_0) > f(0)$。
因为 $f(0) < f(x_0)$，根据条件 ① 必有：
$$D(x_0) \subseteq D(0)$$
我们来具体分析 $D(x_0)$ 与 $D(0)$ 的元素：
- 观察 $D(0)$：由条件 ②，当 $d \in (0, 1)$ 时，有 $f(d) < f(0)$，所以 $d \notin D(0)$。即 $D(0) \cap (0, 1) = \emptyset$。
- 观察 $D(x_0)$（其中 $x_0 < 0$）：对于任意 $d \in (0, -x_0)$，有 $x_0+d < 0$。
  此时由当自变量小于 $0$ 时 $f(x) = 2^x$ 的性质可知：
  $$f(x_0+d) = 2^{x_0+d} > 2^{x_0} = f(x_0)$$
  这说明 $d \in D(x_0)$。因此 $(0, -x_0) \subseteq D(x_0)$。

由于 $x_0 < 0$，有 $-x_0 > 0$。我们在区间 $(0, \min(-x_0, 1))$ 中任取一个数 $d^*$。
- 因为 $0 < d^* < -x_0$，所以 $d^* \in D(x_0)$。由 $D(x_0) \subseteq D(0)$ 得到 $d^* \in D(0)$；
- 另一方面，因为 $0 < d^* < 1$，由条件 ② 得到 $d^* \notin D(0)$。
这产生矛盾！因此，假设不成立，必有 $f(0) \ge 1$。

##### (ii) 证明：$f(x)$ 在区间 $(0, +\infty)$ 单调递增。
我们先证明一个**重要引理**：对任意 $x > 0$，都有 $f(x) \ge 1$。
*引理的证明*：
假设存在某个 $y > 0$ 使得 $f(y) < 1$。
因为当 $x < 0$ 时 $f(x) = 2^x \in (0, 1)$ 且连续，根据介值定理，必然存在唯一的 $x_0 < 0$ 使得 $f(x_0) = 2^{x_0} = f(y)$。
因为 $f(x_0) \le f(y)$，根据条件 ① 必有 $D(y) \subseteq D(x_0)$。
- 对于 $D(x_0)$：若 $d \le 0$，当 $x_0+d < 0$ 时，有 $f(x_0+d) = 2^{x_0+d} \le 2^{x_0} = f(x_0)$；当 $x_0+d \ge 0$ 时，由于 $x_0 < 0$ 必有 $d \ge -x_0 > 0$，这与 $d \le 0$ 矛盾。因此对所有 $d \le 0$，均有 $d \notin D(x_0)$。即 $D(x_0) \cap (-\infty, 0] = \emptyset$。
- 对于 $D(y)$：取 $d = -y < 0$，有 $f(y+d) = f(0) \ge 1 > f(y)$（由假设 $f(y)<1$ 成立）。所以 $-y \in D(y)$。
这导致 $-y \in D(y) \cap (-\infty, 0)$，与 $D(y) \subseteq D(x_0)$ 且 $D(x_0) \cap (-\infty, 0] = \emptyset$ 矛盾！
因此，引理成立，即对任意 $x > 0$，都有 $f(x) \ge 1$。

**第一步：证明 $f(x)$ 在 $(0, +\infty)$ 上单调不减。**
假设存在 $0 < y_1 < y_2$ 使得 $f(y_1) > f(y_2)$。
根据条件 ①，由 $f(y_2) < f(y_1)$ 可得 $D(y_1) \subseteq D(y_2)$。
因为 $f(y_1) \ge 1$，而对任意 $x_0 < 0$ 都有 $f(x_0) = 2^{x_0} < 1 \le f(y_1)$，所以对任意 $x_0 < 0$，都有 $f(x_0) < f(y_1)$。
由条件 ① 可知，对任意 $x_0 < 0$，都有 $D(y_1) \subseteq D(x_0)$。
根据引理证明中的结论，当 $x_0 < 0$ 时有 $D(x_0) \cap (-\infty, 0] = \emptyset$。因此必定有：
$$D(y_1) \cap (-\infty, 0] = \emptyset$$
这说明对任意 $d \le 0$，都有 $d \notin D(y_1)$，即 $f(y_1+d) \le f(y_1)$。
因为 $y_1 > 0$，对任意 $x \in [0, y_1]$，可设 $d = x - y_1 \le 0$，则：
$$f(x) \le f(y_1)$$
同理，因为 $D(y_1) \subseteq D(y_2)$ 且对任意 $x_0 < 0$ 都有 $D(y_2) \subseteq D(x_0)$，故也有 $D(y_2) \cap (-\infty, 0] = \emptyset$。
同理可得，对任意 $x \in [0, y_2]$，有 $f(x) \le f(y_2)$。
因为 $0 < y_1 < y_2$，所以 $y_1 \in [0, y_2]$，从而 $f(y_1) \le f(y_2)$。
这与假设 $f(y_1) > f(y_2)$ 矛盾！
因此，对任意 $0 < y_1 < y_2$，必有 $f(y_1) \le f(y_2)$。即 $f(x)$ 在 $(0, +\infty)$ 上单调不减。

**第二步：证明 $f(x)$ 在 $(0, +\infty)$ 上严格单调递增。**
采用反证法。假设存在 $0 < y_1 < y_2$ 使得 $f(y_1) = f(y_2)$。
由于 $f(x)$ 在 $(0, +\infty)$ 上单调不减，所以对任意 $x \in [y_1, y_2]$，都有 $f(x) = f(y_1)$。
由 $f(y_1) = f(y_2)$，结合条件 ①，有 $D(y_1) = D(y_2)$。
设 $a = y_2 - y_1 > 0$。对任意 $d \in \mathbb{R}$，有：
$$f(y_1+d) > f(y_1) \iff f(y_2+d) > f(y_2) = f(y_1)$$
令 $x = y_1+d$，则上式等价于：对任意 $x \in \mathbb{R}$，
$$f(x) > f(y_1) \iff f(x+a) > f(y_1)$$
这意味着，若存在某个 $x^* \in \mathbb{R}$ 满足 $f(x^*) > f(y_1)$，则对任意 $k \in \mathbb{Z}$ 都有 $f(x^*+ka) > f(y_1)$。
但当 $k$ 取足够小的负整数时， $x^*+ka < 0$，此时有 $f(x^*+ka) = 2^{x^*+ka} < 1 \le f(y_1)$，产生矛盾！
因此，在整个 $\mathbb{R}$ 上不存在任何 $x$ 使得 $f(x) > f(y_1)$。
即对所有的 $x \in \mathbb{R}$，有 $f(x) \le f(y_1)$。
由此可知 $f(y_1)$ 为 $f(x)$ 在整个 $\mathbb{R}$ 上的最大值，因此 $f(0) \le f(y_1)$。

根据条件 ②，当 $x \in (0, 1)$ 时，有 $f(x) < f(0)$。
所以当 $x \in (0, 1)$ 时， $f(x) < f(0) \le f(y_1)$。
我们选取一个特定的 $x_1 \in (0, 1)$。
- 因为 $f(x_1) < f(0)$，取 $d = -x_1 < 0$，有 $f(x_1+d) = f(0) > f(x_1)$。所以 $-x_1 \in D(x_1)$。
- 另一方面，因为 $x_1 > 0$，由引理有 $f(x_1) \ge 1$。
  对任意 $x_0 < 0$，有 $f(x_0) = 2^{x_0} < 1 \le f(x_1)$。由条件 ① 得到 $D(x_1) \subseteq D(x_0)$。
  由前面的证明知，当 $x_0 < 0$ 时有 $D(x_0) \cap (-\infty, 0] = \emptyset$。
  从而 $D(x_1) \cap (-\infty, 0] = \emptyset$。这要求 $D(x_1)$ 中不含负数。

由于 $-x_1 < 0$ 且 $-x_1 \in D(x_1)$，这与 $D(x_1) \cap (-\infty, 0] = \emptyset$ 矛盾！
因此，不存在 $0 < y_1 < y_2$ 使得 $f(y_1) = f(y_2)$。
综上所述，$f(x)$ 在 $(0, +\infty)$ 内是严格单调递增的。
