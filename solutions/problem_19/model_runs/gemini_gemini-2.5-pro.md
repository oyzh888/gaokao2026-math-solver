(1)
根据题意，当 `x<0` 时，`f(x)=2^x`；当 `x≥0` 时，`f(x)=1-x`。
我们要求解的集合是 `D(-1) = { d∈R : f(-1+d) > f(-1) }`。

首先，计算 `f(-1)` 的值。因为 `-1<0`，所以 `f(-1) = 2^{-1} = 1/2`。
于是，我们需要求解不等式 `f(-1+d) > 1/2`。
对变量 `-1+d` 的取值范围进行分类讨论：

情况一：`-1+d < 0`，即 `d < 1`。
此时，`f(-1+d) = 2^{-1+d}`。
不等式为 `2^{-1+d} > 1/2`，即 `2^{-1+d} > 2^{-1}`。
由于指数函数 `y=2^t` 是增函数，所以 `-1+d > -1`，解得 `d > 0`。
综合本情况的条件 `d<1`，可得 `0 < d < 1`。

情况二：`-1+d ≥ 0`，即 `d ≥ 1`。
此时，`f(-1+d) = 1 - (-1+d) = 2-d`。
不等式为 `2-d > 1/2`。
解这个不等式，得 `d < 2 - 1/2`，即 `d < 3/2`。
综合本情况的条件 `d≥1`，可得 `1 ≤ d < 3/2`。

综上所述，`d` 的取值范围是 `(0, 1) ∪ [1, 3/2)`，即 `(0, 3/2)`。
所以，`D(-1) = (0, 3/2)`。

**最终结论：(1) D(-1) = (0, 3/2)。**

(2)
由于 `f(x)` 是奇函数，且当 `x<0` 时 `f(x)=2^x`，我们可以确定 `f(x)` 的完整表达式。
当 `x > 0` 时，`-x < 0`，则 `f(x) = -f(-x) = -2^{-x}`。
当 `x = 0` 时，`f(0) = -f(-0)`，即 `2f(0)=0`，所以 `f(0)=0`。
综上，函数的表达式为：
`f(x) =`
`  2^x,     x < 0`
`  0,       x = 0`
`  -2^{-x},  x > 0`

要证明 `D(x₂) ⊆ D(x₁)`，我们只需证明对任意 `d ∈ D(x₂)`，必有 `d ∈ D(x₁)`。
设 `d ∈ D(x₂)`，由定义可知 `f(x₂ + d) > f(x₂)`。我们的目标是证明 `f(x₁ + d) > f(x₁)`。

我们先分析函数 `f(x)` 的单调性。
当 `x < 0` 时，`f(x) = 2^x` 是严格增函数。
当 `x > 0` 时，`f(x) = -2^{-x}` 也是严格增函数（因为 `-x` 递减，`2^{-x}` 递减，`-2^{-x}` 递增）。
函数在 `(-∞, 0)` 和 `(0, +∞)` 上分别严格单调递增。

现在，我们利用条件 `f(x₁) ≤ f(x₂)` 和 `x₁·x₂ ≠ 0`，对 `x₁` 和 `x₂` 的正负性进行分类讨论。

情况一：`x₁ > 0` 且 `x₂ > 0`。
因为 `f(x)` 在 `(0, +∞)` 上严格递增，所以 `f(x₁) ≤ f(x₂)` 蕴含 `x₁ ≤ x₂`。
要证明 `D(x₂) ⊆ D(x₁)`，即证明：
`(-∞, -x₂] ∪ (0, +∞) ⊆ (-∞, -x₁] ∪ (0, +∞)`。（*推导见附注）
因为 `x₁ ≤ x₂`，所以 `-x₁ ≥ -x₂`，故 `(-∞, -x₂] ⊆ (-∞, -x₁]`。
因此，`(-∞, -x₂] ∪ (0, +∞) ⊆ (-∞, -x₁] ∪ (0, +∞)` 成立。

情况二：`x₁ < 0` 且 `x₂ < 0`。
因为 `f(x)` 在 `(-∞, 0)` 上严格递增，所以 `f(x₁) ≤ f(x₂)` 蕴含 `x₁ ≤ x₂`。
要证明 `D(x₂) ⊆ D(x₁)`，即证明 `(0, -x₂) ⊆ (0, -x₁)`。（*推导见附注）
因为 `x₁ ≤ x₂ < 0`，所以 `-x₁ ≥ -x₂ > 0`。
因此，`(0, -x₂) ⊆ (0, -x₁)` 成立。

情况三：`x₁` 和 `x₂` 异号。
如果 `x₁ > 0, x₂ < 0`：此时 `f(x₁) = -2^{-x₁} < 0`，而 `f(x₂) = 2^{x₂} > 0`。所以 `f(x₁) < f(x₂)`，满足 `f(x₁) ≤ f(x₂)`。
我们需要证明 `D(x₂) ⊆ D(x₁)`。
`D(x₂) = (0, -x₂)`。
`D(x₁) = (-∞, -x₁] ∪ (0, +∞)`。
因为 `x₂ < 0`，所以 `-x₂ > 0`，区间 `(0, -x₂)` 是 `(0, +∞)` 的子集。
因此，`D(x₂) ⊆ D(x₁)` 成立。
如果 `x₁ < 0, x₂ > 0`：此时 `f(x₁) = 2^{x₁} > 0`，而 `f(x₂) = -2^{-x₂} < 0`。所以 `f(x₁) > f(x₂)`，不满足 `f(x₁) ≤ f(x₂)` 的条件。此种情形不必考虑。

综上所述，在所有满足条件的情形下，均有 `D(x₂) ⊆ D(x₁)`。

*附注：`D(x₀)` 的推导
若 `x₀ < 0`，解 `f(x₀+d) > 2^{x₀}`。`f(x₀+d)` 的值大于一个正数，所以 `x₀+d` 必须小于 0。即 `d < -x₀`。此时 `f(x₀+d)=2^{x₀+d}`，不等式化为 `2^{x₀+d} > 2^{x₀}`，得 `d > 0`。故 `D(x₀) = (0, -x₀)`。
若 `x₀ > 0`，解 `f(x₀+d) > -2^{-x₀}`。
- 若 `x₀+d > 0`，`-2^{-(x₀+d)} > -2^{-x₀}`，得 `2^{-(x₀+d)} < 2^{-x₀}`，即 `-(x₀+d) < -x₀`，解得 `d > 0`。
- 若 `x₀+d = 0`，`f(0) > -2^{-x₀}`，即 `0 > -2^{-x₀}`，恒成立。故 `d=-x₀` 满足。
- 若 `x₀+d < 0`，`2^{x₀+d} > -2^{-x₀}`，左边为正，右边为负，恒成立。故所有 `d < -x₀` 均满足。
合并得到 `D(x₀) = (-∞, -x₀] ∪ (0, +∞)`。

**最终结论：(2) 证明如上，结论成立。**

**(i) 证明：f(0) ≥ 1**
采用反证法。假设 `f(0) < 1`。
由 `x<0` 时 `f(x) = 2^x` 可知，`lim_{x→0⁻} f(x) = lim_{x→0⁻} 2^x = 1`。
因为 `f(0) < 1`，根据极限的定义，存在 `x₁ < 0` 使得 `f(0) < f(x₁) < 1`。
（例如，取 `y = (f(0)+1)/2`，则 `f(0)<y<1`。由于 `f(x)=2^x` 在 `(-∞,0)` 的值域是 `(0,1)`，必存在 `x₁<0` 使得 `f(x₁)=y`。）
我们找到了 `x₁ < 0` 满足 `f(x₁) > f(0)`。
根据条件① `若 f(x_a) ≤ f(x_b)，则 D(x_b) ⊆ D(x_a)`，令 `x_a=0, x_b=x₁`，则有 `D(x₁) ⊆ D(0)`。
现在分析 `D(x₁)`。因为 `x₁ < 0`，`f(x)` 在 `x₁` 的一个邻域内是 `f(x)=2^x`，是严格增函数。
取一个充分小的正数 `d`，例如 `d ∈ (0, -x₁)`，则 `x₁+d < 0`。
`f(x₁+d) = 2^{x₁+d} > 2^{x₁} = f(x₁)`。
这意味着所有满足 `0 < d < -x₁` 的 `d` 都在 `D(x₁)` 中，即 `(0, -x₁) ⊆ D(x₁)`。
因为 `D(x₁) ⊆ D(0)`，所以 `(0, -x₁) ⊆ D(0)`。
由 `D(0)` 的定义，对任意 `d ∈ (0, -x₁)`，都有 `f(0+d) > f(0)`，即 `f(d) > f(0)`。
因为 `x₁ < 0`，所以 `-x₁ > 0`。这意味着我们找到了一个正数区间 `(0, -x₁)`，在此区间内 `f(d) > f(0)`。
然而，条件②指出，当 `0 < x < 1` 时，`f(x) < f(0)`。
我们可以在 `(0, -x₁) ∩ (0, 1)` 中取值（此交集非空），从而得到矛盾。例如，取 `d_0 = min(-x₁/2, 1/2)`，则 `d_0 > 0` 且 `d_0` 同时属于 `(0, -x₁)` 和 `(0,1)`。
一方面，`f(d_0) > f(0)`。
另一方面，`f(d_0) < f(0)`。
矛盾出现。因此，最初的假设 `f(0) < 1` 不成立。
故 `f(0) ≥ 1`。

**(ii) 证明：f(x) 在区间 (0,+∞) 单调递增**
我们需要证明，对任意 `0 < x₁ < x₂`，都有 `f(x₁) < f(x₂)`。
我们使用反证法，假设存在 `0 < x₁ < x₂` 使得 `f(x₁) ≥ f(x₂)`。

首先证明 `f(x)` 在 `(0,1)` 上严格单调递增。
假设存在 `0 < a < b < 1` 使得 `f(a) ≥ f(b)`。
由条件①，`D(b) ⊆ D(a)`。
由条件②，因为 `b ∈ (0,1)`，所以 `f(b) < f(0)`。
考虑 `d = -b`，则 `f(b+d) = f(0) > f(b)`，因此 `-b ∈ D(b)`。
由于 `D(b) ⊆ D(a)`，必有 `-b ∈ D(a)`。
这意味着 `f(a+(-b)) > f(a)`，即 `f(a-b) > f(a)`。
设 `x_a = a`，`x_b = a-b`。因为 `0 < a < b < 1`，所以 `a-b < 0`。
我们得到 `f(x_b) > f(x_a)`，其中 `x_b < 0`, `x_a ∈ (0,1)`。
再次应用条件①，`f(x_a) < f(x_b)` 蕴含 `D(x_b) ⊆ D(x_a)`。
我们已知 `-b ∈ D(a)`，所以现在有 `-b ∈ D(x_b)`，即 `-b ∈ D(a-b)`。
这意味着 `f((a-b)+(-b)) > f(a-b)`，即 `f(a-2b) > f(a-b)`。
`a-b` 和 `a-2b` 都是负数。所以 `f(a-2b) = 2^{a-2b}`，`f(a-b) = 2^{a-b}`。
不等式化为 `2^{a-2b} > 2^{a-b}`。
因为指数函数 `y=2^t` 严格递增，所以 `a-2b > a-b`，即 `-2b > -b`。
两边同乘-1，得 `2b < b`，这意味着 `b < 0`。
这与我们假设的 `b > 0` 矛盾。
因此，`f(x)` 在 `(0,1)` 上必须是严格单调递增的。

接下来，利用 `f` 在 `(0,1)` 上的单调性来证明其在整个 `(0,+∞)` 上的单调性。
假设存在 `0 < x₁ < x₂` 使得 `f(x₁) ≥ f(x₂)`。
由条件①，`D(x₂) ⊆ D(x₁)`.
取一个 `y ∈ (0,1)` 且 `y < x₂ - x₁`。
因为我们已经证明 `f` 在 `(0,1)` 上严格递增，并且由(i)的结论，`\forall x \in (0,1), f(x) < f(0)`，我们可以得到，对任意 `y' \in (0,y)`，有 `f(y') < f(y)`。
由条件①，`D(y) subseteq D(y')`。
由条件②，`f(y) < f(0)`，所以 `-y \in D(y)`。
因此，`\forall y' \in (0,y)`，都有 `-y \in D(y')`。
这意味着 `f(y' - y) > f(y')`。即 `2^{y'-y} > f(y')`。

回到我们的主要假设 `f(x₁) \ge f(x₂)`。
现在，我们构造一个序列。
因为 `f` 在 `(0,1)` 上严格递增，且 `x_1>0`，所以 `\forall y \in (0, \min(1, x_1))`，都有 `f(y) < f(x_1)`（若 `x_1<1`）或 `f(y) < \sup_{z\in(0,1)}f(z) \le f(x_1)`（若 `x_1\ge 1`，证明见后）。总之 `f(y) < f(x_1)`。
由条件①，`D(x₁) ⊆ D(y)`。
因为 `-y \in D(y)`，所以 `-y \in D(x₁)`。
这意味着 `f(x₁-y) > f(x₁)`。
这与 `f` 的单调性相矛盾：`x₁-y < x₁`，但函数值却增大了。
我们来构造一个矛盾序列：
令 `y_0 = x_1`。我们有 `f(y_0 - y) > f(y_0)`。
令 `y_1 = y_0 - y = x_1 - y`。我们有 `f(y_1) > f(y_0)`。
由于 `y_0 > 0`，只要 `y` 足够小，`y_1` 仍然大于 `y`。
只要 `f(y_k) > f(y)`，我们就可以重复这个步骤：`D(y_k) \subseteq D(y) \implies -y \in D(y_k) \implies f(y_k - y) > f(y_k)`。
令 `y_k = x_1 - ky`。只要 `y_{k-1}>y`，我们就有 `f(y_k) > f(y_{k-1})`。
这个递推可以进行 `K = \lfloor (x_1-y)/y \rfloor` 次。
我们得到一个序列 `x_1, x_1-y, x_1-2y, ..., x_1-(K+1)y`。
其中 `y_K = x_1-Ky > y`，`y_{K+1} = x_1-(K+1)y \in (0, y]`。
我们有 `f(y_{K+1}) > f(y_K) > ... > f(y_1) > f(y_0) = f(x_1)`。
所以 `f(y_{K+1}) > f(x_1)`。
但我们又有 `0 < y_{K+1} \le y < \min(1, x_1) \le x_1`。
如果 `x_1 < 1`，则 `y_{K+1} < x_1`。因为 `f` 在 `(0,1)` 上严格递增，所以 `f(y_{K+1}) < f(x_1)`。这与 `f(y_{K+1}) > f(x_1)` 矛盾。
如果 `x_1 \ge 1`，我们先证明 `\forall z \in (0,1), f(z) < f(x_1)`。假设存在 `z \in (0,1)` 使得 `f(z) \ge f(x_1)`。则 `D(x_1) \subseteq D(z)`。`-z \in D(z)`，所以 `-z \in D(x_1)`。`f(x_1-z) > f(x_1)`。但 `x_1-z < x_1`，这构成了一个“下降点”。重复这个过程，`x_k = x_1-kz`，最终会落入 `(0,1)` 区间，并与 `f` 在 `(0,1)` 的单调性矛盾。
所以 `\forall z \in (0,1), f(z) < f(x_1)` 成立。
`y_{K+1} \in (0,1)`，所以 `f(y_{K+1}) < f(x_1)`。这与 `f(y_{K+1}) > f(x_1)` 矛盾。
因此，最初的假设 `f(x₁) ≥ f(x₂)` 不成立。
故 `f(x)` 在 `(0, +∞)` 上严格单调递增。

**最终结论：(3) (i) 证明如上，f(0)≥1；(ii) 证明如上，f(x)在(0,+∞)单调递增。**
