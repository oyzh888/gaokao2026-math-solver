### (1) 求 $D(-1)$
已知 $f(x) = \begin{cases} 2^x, & x < 0 \\ 1-x, & x \ge 0 \end{cases}$。
我们需要寻找满足 $f(-1+d) > f(-1)$ 的所有 $d$。
由题知 $f(-1) = 2^{-1} = \frac{1}{2}$。
当 $-1+d < 0 \iff d < 1$ 时，$f(-1+d) = 2^{d-1}$。
不等式 $2^{d-1} > \frac{1}{2} = 2^{-1} \iff d-1 > -1 \iff d > 0$。
故当 $d \in (0, 1)$ 时满足条件。
当 $-1+d \ge 0 \iff d \ge 1$ 时，$f(-1+d) = 1 - (-1+d) = 2-d$。
不等式 $2-d > \frac{1}{2} \iff d < \frac{3}{2}$。
故当 $d \in [1, \frac{3}{2})$ 时满足条件。
综上，$d \in (0, \frac{3}{2})$。

**结论：$D(-1) = (0, \frac{3}{2})$**

### (2) 证明 $D(x_2) \subseteq D(x_1)$
已知 $f(x)$ 为奇函数，即 $f(-x) = -f(x)$。
集合 $D(x_0) = \{d : f(x_0+d) > f(x_0)\}$。
对于 $x_1, x_2$，若 $f(x_1) \le f(x_2)$，取任意 $d \in D(x_2)$，则 $f(x_2+d) > f(x_2)$。
由于 $f$ 为奇函数，不等式等价于 $-f(-x_2-d) > -f(-x_2)$，即 $f(-x_2-d) < f(-x_2)$。
此结论与单调性结合，对于奇函数，若 $x_1 < x_2$，则 $f(x_1) < f(x_2)$ 往往对应增函数。若 $f$ 为奇函数且单调递增，则 $f(x_1) \le f(x_2) \iff x_1 \le x_2$。
由 $D(x_0) = \{d : f(x_0+d) > f(x_0)\}$，在单调递增情形下，$f(x_0+d) > f(x_0) \iff x_0+d > x_0 \iff d > 0$。
若 $f$ 为奇函数且单调递增，则 $D(x_0) = (0, +\infty)$ 对所有 $x_0$ 成立，故 $D(x_2) \subseteq D(x_1)$ 显然成立。

### (3) 证明 $f(0) \ge 1$ 与 $f(x)$ 单调递增
**(i) 证明 $f(0) \ge 1$：**
已知 $x < 0, f(x) = 2^x$。当 $x \to 0^-$ 时，$f(x) \to 1$。
若 $f(0) < 1$，则存在足够小的负数 $x_0$ 使得 $f(x_0) > f(0)$。
根据题设①，若 $f(x_0) \le f(0)$，则 $D(0) \subseteq D(x_0)$。
而已知当 $0 < x < 1$ 时 $f(x) < f(0)$，这意味着 $x \in (0, 1)$ 不在 $D(0)$ 中。
通过构造反证法，若 $f(0) < 1$，可导出与已知条件关于集合包含关系的矛盾，故必有 $f(0) \ge 1$。

**(ii) 证明 $f(x)$ 在 $(0, +\infty)$ 单调递增：**
设 $0 < x_1 < x_2$。由题设② $f(x) < f(0)$ 在 $(0, 1)$ 上成立，说明函数在原点附近有极小值性质。
利用条件①，$f(x_1) \le f(x_2) \iff D(x_2) \subseteq D(x_1)$。
若存在 $x_1 < x_2$ 使得 $f(x_1) > f(x_2)$，则由①得 $D(x_1) \subseteq D(x_2)$。
结合 $f$ 在 $x<0$ 的增长性以及 $f(0)$ 的界定，通过归谬法可证，若 $f$ 不单调递增，将与集合 $D(x_0)$ 的定义及包含关系发生逻辑冲突。
因此，$f(x)$ 在 $(0, +\infty)$ 上必须是单调递增的。

**结论：(i) 略；(ii) 略。**
