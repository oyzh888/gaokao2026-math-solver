# 🎓 Gaokao 2026 Math Solver

用 **Claude Code** 去解 2026 年中国高考考生最多卷种（新高考 I 卷）数学的压轴题（第 19 题），并把全过程开源。

## 这是什么
- **卷种**：2026 新高考 I 卷（11 省采用：浙江、山东、广东、福建、湖南、湖北、河北、江苏、江西、安徽、河南——考生规模最大）。
- **目标题**：第 19 题（17 分），函数 + 集合 `D(x)` 的"新题型"分析题，被评为本卷最难、逼近全国卷历史难度峰值。
- **玩法**：把题目喂给 Claude Code，记录它的推理、解答、自我验证全过程，沉淀成可复现的开源仓库。

试卷为考后**网络整理版**（来源：高考100 等教育门户），**非教育部官方发布**。官方仅在 6/23 起发布"试题评析"（不含逐题标答）。题面与答案请交叉核对，仅供学习研究。本仓库不含任何付费/受版权保护的教辅内容，仅汇总公开网络资源链接与自动求解过程。
