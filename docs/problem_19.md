## 背景
2026 新高考 I 卷由教育部统一命制，使用省份 11 个：浙江、山东、广东、福建、湖南、湖北、河北、江苏、江西、安徽、河南——是覆盖省份最多、考生规模最大的卷种。第 19 题是全卷压轴题，被一线教师评为"今年最震撼的一道题"，难度逼近全国卷历史峰值。

## 题目（事实性转述）
设函数 `f(x)` 定义在 R 上。对 `x ∈ R`，定义集合

    D(x) = { t ∈ R : f(x + t) < f(x) }

即把自变量平移 `t` 后函数值变小（变大的反向）的那些平移量的集合。

围绕该集合，题目分三问（17 分）：

- **(1)** 给定具体分段函数，求某点处的集合 `D(x)`，并据此判断单调性相关结论。（读懂定义即可）
- **(2)** 在给定条件下，讨论 `D(x)` 的结构（如对一切 x 是否有 `0 < D(x)` 等包含关系），需要分类讨论。被认为是失分重灾区。
- **(3)** 证明题：结合奇函数性质与中间结论"`f(x)` 在 `x>0` 时非正"，证明 `f` 在 `(0, +∞)` 上的单调性结论。极难。
